# social-content

Write ready-to-shoot short-form scripts without touching product source.

## Flow

1. Read product positioning from `product-marketing.md`, README, or supplied context.
2. Identify audience, promise, activation moment, and voice.
3. Produce hooks, script beats, on-screen text, caption, CTA, and production notes.
4. Group scripts by angle or funnel stage when making a batch.
5. Save to a local markdown artifact only when useful.

## Rules

- Do not invent customer proof, metrics, or testimonials.
- Do not post, schedule, or call social APIs.
- Keep scripts shootable with realistic props and screen recordings.
- Label inferred positioning when no source exists.

## Output

```text
STATUS: DONE | DONE_WITH_CONCERNS | NEEDS_CONTEXT | BLOCKED
VERDICT: SCRIPTS
SUMMARY: <scripts/angles produced>
NEXT: shoot, revise, or none
```
