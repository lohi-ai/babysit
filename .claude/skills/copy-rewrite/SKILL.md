# copy-rewrite
Make marketing copy clearer, more specific, and closer to the product's
actual positioning. Ground in `product-marketing.md`, README, or the existing
page; rewrite only the requested or highest-impact text, in place. Never
invent claims, customer names, numbers, integrations, or outcomes; preserve
brand voice unless asked to change it. Verify the text renders and fits when
a UI target is available; layout changes stay out of scope unless needed for
text fit.
## Output
```text
STATUS: DONE | DONE_WITH_CONCERNS | NEEDS_CONTEXT | BLOCKED
VERDICT: REWRITTEN
SUMMARY: <copy changed + verification>
NEXT: create-pr, conversion-fix, or none
```
